::: tip Python urllib
urllib 是 Python 内置的 HTTP 请求库，无需额外安装即可使用。
::: right
[官方文档](https://docs.python.org/library/urllib.html)
:::

::: warning ⚠️注意
在 Python 2 中，有 urllib 和 urllib2 两个库来实现 HTTP 请求，而在 Python 3 中，统一为 urllib。
:::
